# Feira Fácil 🧺🍃

Sua feira inteligente em família. O **Feira Fácil** é um aplicativo mobile premium desenvolvido em Flutter para organizar, comparar e economizar nas suas compras de feira e supermercado.

![Feira Fácil Logo](logo.png)

## ✨ Funcionalidades Premium

- **💰 Inteligência de Preços**: O app aprende com o seu histórico. Saiba instantaneamente se o preço atual de um item está mais caro ou mais barato do que na última vez que o grupo comprou.
- **👥 Gestão de Grupos**: Compartilhe suas listas em tempo real com sua família através de códigos de convite.
- **📊 Dashboard Dinâmico**: Acompanhe o progresso da sua feira com barras de progresso visuais e indicadores de preço estimado vs. atual.
- **🏷️ Categorização Automática**: Itens organizados por seções para agilizar sua caminhada no mercado.

## 🚀 Tecnologias

- **Framework**: [Flutter](https://flutter.dev) (v3.x)
- **State Management**: [Riverpod 3.0](https://riverpod.dev)
- **Backend / Database**: [Firebase](https://firebase.google.com) (Firestore & Auth)
- **Architecture**: Domain-Driven Design (DDD) simplificado para escalabilidade.

## 🛠️ Como rodar o projeto

1.  Certifique-se de ter o Flutter instalado (`flutter doctor`).
2.  Clone o repositório: `git clone https://github.com/gustavoernandes86/feira_facil.git`
3.  Instale as dependências: `cd feira_facil && flutter pub get`
4.  Configure o seu projeto no Firebase e baixe os arquivos de configuração (google-services.json para Android).
5.  Execute o app: `flutter run`
